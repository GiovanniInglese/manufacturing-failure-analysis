#  Manufacturing Process Performance & Failure Analysis

## Project Overview

In this project, I analyzed two manufacturing processes to evaluate their performance, consistency, and reliability. The goal was to go beyond simple averages and identify **what drives machine failures** and how operating conditions impact production quality.

Using Python for statistical analysis and Tableau for visualization, I explored how factors like torque and tool wear influence failure rates and overall process stability.

---

##  Tools & Technologies

- Python (Pandas, NumPy)
- Statistical Analysis (Hypothesis Testing, Correlation)
- Tableau (Interactive Dashboard)
- Data Cleaning & Feature Engineering

---

##  What I Did

### 1. Data Preparation
- Loaded and cleaned the dataset
- Standardized column names for easier analysis
- Checked for missing values and validated data types

---

### 2. Process Comparison (Hypothesis Testing)
- Compared average torque between two processes
- Performed statistical testing to determine if differences were significant
- Found that both processes produce **nearly identical average output**

---

### 3. Variability & Reliability Analysis
- Analyzed standard deviation to measure consistency
- Compared failure rates between processes
- Identified that **Process 2 is more reliable**, with:
  - Lower variability
  - Fewer machine failures

---

### 4. Failure Rate Analysis
- Calculated failure rates instead of just counting failures
- Segmented data by torque levels and tool wear
- Identified patterns in when failures occur

---

### 5. Failure Driver Investigation
- Performed correlation analysis to identify key drivers of failure
- Found that:
  - **Torque** is the strongest contributor
  - **Tool wear** is a secondary but important factor

---

### 6. Risk Threshold Detection (Key Insight)
- Identified operational conditions where failure probability spikes
- Defined “high-risk” scenarios based on:
  - High torque levels
  - Excessive tool wear



## Key Insights

- Both processes produce similar average torque (~40 Nm)
- Process 2 is significantly more consistent and reliable
- Failure rates increase sharply under:
  - High torque conditions
  - High tool wear levels
- Failures are **not random** — they are driven by operating conditions
- Medium torque ranges represent the **optimal operating zone**

---

##  Business Recommendations

- Monitor and control torque levels to prevent excessive strain
- Implement proactive maintenance based on tool wear thresholds
- Prioritize Process 2 for more stable production
- Use identified risk conditions to reduce downtime and improve efficiency
